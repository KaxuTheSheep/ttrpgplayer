Shields are defensive tools used to **redirect**, **absorb**, or **prevent** incoming strikes.  
Each shield type defines how you defend:  
some favour **agility**, others **durability**, and others **area control**.
All Shields take up a hand slot, aside from the Buckler.
Shields modify **[[Block]]** and **[[Parry]]**, but do not affect melee [[Damage Types|damage]] unless specified.

Each shield has:
- A **[[Block]] Die** (how much [[Damage Types|damage]] it can prevent when Blocking)
- A **[[Parry]] Adjustment** (how it affects finesse defence)
- A **[[Speed & Resolution| Speed]]** (an addition used during blocking and parrying)
- An **[[Evasion]] Modifier** (large shields make you easier to hit)

### Buckler
_A small fist-shield used to deflect rather than absorb._ 

| Property          | Effect                                 |
| ----------------- | -------------------------------------- |
| **[[Block]] Die**     | **1d4** per [[Block]]                      |
| **[[Parry]]**         | **Advantage**                          |
| **[[Evasion]]**       | No change                              |
| **[[Speed & Resolution\|Speed]]**         | +2                                     |
| **Weight / Bulk** | Light — does **not** restrict [[Movement Types\|movement]] |
### Heater
_A balanced defensive shield offering reliable protection._

| Property                          | Effect                                                       |
| --------------------------------- | ------------------------------------------------------------ |
| **[[Block]] Die**                 | **1d8** per [[Block]]                                        |
| **[[Parry]]**                     | No bonus or penalty                                          |
| **[[Evasion]]**                   | No change                                                    |
| **[[Speed & Resolution\|Speed]]** | 0                                                            |
| **Weight / Bulk**                 | Standard — can be used in one hand without major restriction |
### Pavise
_A massive shield used to cover the body and control space._

| Property          | Effect                                                        |
| ----------------- | ------------------------------------------------------------- |
| **[[Block]] Die**     | **1d12** per [[Block]]                                            |
| **[[Parry]]**         | **Disadvantage**                                              |
| **[[Evasion]]**       | **-2**                                                        |
| **[[Speed & Resolution\|Speed]]**         | -2                                                            |
| **Weight / Bulk** | Heavy — may restrict [[Movement Types\|movement]] and positioning in tight spaces |
