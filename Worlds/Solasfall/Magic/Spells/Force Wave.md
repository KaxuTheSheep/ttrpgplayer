
## Process
1. Draw ambient mana
2. Amplify kinetic energy in a spherical pattern around caster
3. Release energy outward as a shockwave
4. Release mana

## Limitations
- Affects all creatures within radius (including allies)
- Pushes creatures back, dealing damage on collision
- Can be used defensively to break grapples

## Variables

| Variable | Base | Scaling |
|----------|------|---------|
| Damage | 1d4 bludgeoning | +1 die step per point |
| Radius | 5m | +2m per point |
| Push Distance | 3m | +2m per point |
| Action Cost | 2 Actions | -0.5 Actions per point |
| Priority | 6 + Grace | +2 priority per point |

## Flux Casting

| Stat | Value |
|------|-------|
| Control DN | 8 + Enhancement |
| Manifest DN | 12 + (Enhancement × 2) |

**Backlash Examples:**
- Wave inverts: pulls creatures toward caster instead
- Wave collapses: deals damage to caster only
- Directional rupture: wave only goes in one direction (GM determines)

## Weave Casting

| Stat | Value |
|------|-------|
| Time | 8 + Enhancement (draw time) |
| Defect DN | 12 + (Enhancement × 2) |

**Defect Examples:**
- Radius smaller by half
- Damage reduced by half
- Push distance is random (1d4m)
