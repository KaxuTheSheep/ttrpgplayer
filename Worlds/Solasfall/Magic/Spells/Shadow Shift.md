
## Process
1. Draw ambient mana
2. Suppress all light wavelengths in target area
3. Maintain suppression field
4. Release mana

## Limitations
- No light penetrates the area
- Creatures inside cannot see
- Creatures outside cannot see in
- Does not block other senses (sound, smell)

## Variables

| Variable | Base | Scaling |
|----------|------|---------|
| Area Radius | 5m | +2m per point |
| Duration | 2 rounds | +1 round per point |
| Mobility | Static | 2 points: mobile with caster (radius halved) |
| Priority | 5 + Grace | +2 priority per point |

## Flux Casting

| Stat | Value |
|------|-------|
| Control DN | 7 + Enhancement |
| Manifest DN | 11 + (Enhancement × 2) |

**Backlash Examples:**
- Darkness flickers: creatures inside can see for 1 Tick each round
- Darkness collapses: blinds the caster for 1 round
- Light inversion: bright flash blinds everyone in 10m for 1 round

## Weave Casting

| Stat | Value |
|------|-------|
| Time | 7 + Enhancement (draw time) |
| Defect DN | 11 + (Enhancement × 2) |

**Defect Examples:**
- Area reduced by half
- Duration reduced by half
- Darkness only suppresses some light wavelengths (dim, not dark)
